This project is a **Three.js-powered 3D scene** featuring **high-quality lighting, bloom effects, and camera controls** for an immersive experience. The scene is rendered with realistic light interactions, shadows, and a post-processing bloom effect to enhance the visuals.  

## ‚ú® Features  

- **Three.js** ‚Äì 3D rendering engine  
- **GLTFLoader** ‚Äì Loads `.gltf` models into the scene  
- **OrbitControls** ‚Äì Enables smooth camera movement and zooming  
- **EffectComposer** ‚Äì Post-processing pipeline  
- **UnrealBloomPass** ‚Äì Applies bloom effects for a cinematic glow  
- **Tone Mapping** ‚Äì ACESFilmicToneMapping for realistic color grading  
- **Multiple Light Sources** ‚Äì Ambient, Directional, and Point lights for enhanced realism  

## üì¶ Libraries & Tools Used  

- [Three.js](https://threejs.org/) ‚Äì Core rendering engine  
- [GLTFLoader](https://threejs.org/docs/#examples/en/loaders/GLTFLoader) ‚Äì To import `.gltf` models
